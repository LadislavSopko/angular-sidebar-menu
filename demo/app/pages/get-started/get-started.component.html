Getting Started

This guide explains how to setup your project to begin using Angular Sidebar Menu. It includes information on prerequisites and installing Angular Sidebar Menu.

Download

Angular Admin LTE is available at npm, if you have an existing application run the following command to download it to your project. Angular Sidebar Menu is distributed in Angular Package Format format.

npm install angular-admin-lte --save

Dependencies

Angular Sidebar Menu has no JS dependencies but still css for icons and themes. Import them to your application by using either angular.json or @import in your global app styles file.

"styles": [ "node_modules/font-awesome/css/font-awesome.css", "node_modules/angular-sidebar-menu/themes/admin-lte.scss", "demo/styles.scss" ] @import '~font-awesome/css/font-awesome.css'; @import '~angular-sidebar-menu/themes/admin-lte.scss';

Import

Once Angular Sidebar Menu is downloaded and configured, module can be imported from 'angular-sidebar-menu' shorthand in your application code.

{{ importLib }}

Display a menu

Once Angular Sidebar Menu module is imported you will need to call the Angular Sidebar Menu component in a template. Now you can start configuring the menu, see the Configuration page.

{{ callComponent }}
